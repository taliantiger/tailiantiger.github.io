Z字形变换

将一个给定字符串根据给定的行数，以从上往下、从左到右进行 Z 字形排列。

比如输入字符串为 “LEETCODEISHIRING” 行数为 3 时，排列如下：

L     C     I       R
E T O E S   I    I   G
E     D    H      N

之后，你的输出需要从左往右逐行读取，产生出一个新的字符串，比如：“LCIRETOESIIGEDHN”。

请你实现这个将字符串进行指定行数变换的函数：

string convert(string s, int numRows); 示例 1:

输入: s = “LEETCODEISHIRING”, numRows = 3 输出: “LCIRETOESIIGEDHN” 示例 2:

输入: s = “LEETCODEISHIRING”, numRows = 4 输出: “LDREOEIIECIHNTSG” 解释:

L         D           R
E     O E       I    I
E C       I H       N
T          S          G

  • 解题思路:

    只需创建一维数组，利用各个元素，存储各行排列的结果（第0个元素存储第0行结果，第1个元素存储第1行结果…）；

    巧妙利用flag，改变向上向下遍历的方向。

class Solution {
	// 定义字符串s，定义数字numRows代表字符串拍好后的最大行数的值。
    public String convert(String s, int numRows) {
		// 如果最大行数小于2，则返回字符串s
        if(numRows < 2) return s;
		//  ArrayList是实现了List接口，new ArrayList就是创建来一个数组的意思
		// 声明rows是一个数组，是StringBuilder类（list代表集合，new ArrayList代表建立数组）

		// 本地运行说找不到符号，符号Arraylist，但可以网上提交，难道本地不支持集合List？？？
		// 还是要import什么文件？？？
        List<StringBuilder> rows = new ArrayList<StringBuilder>();
		// 当i小于最大行数值时，不断给创建rows的新元素，同时list集合。
        for(int i = 0; i < numRows; i++) rows.add(new StringBuilder());
		// 巧妙运用flag，改变执行方向。
        int i = 0, flag = -1;

		// toCharArray() 方法：将字符串转换为字符数组
		// 从 c 开始遍历字符数组s的每一个元素。
        for(char c : s.toCharArray()) {
			// array.get() ：从一个数组中得到相应下标的元素
			// 从rows中获得i下标的元素，再给这个元素最后添加“ c”
            rows.get(i).append(c);
			// 如果到达rows元素的最上面或最下面
            if(i == 0 || i == numRows -1)
				// 改变flag的符号
				flag = - flag;
				// 如果在最上面，就不断相加，在最下面，就不断相减
            i += flag;
        }

		// 初始化一个可变字符串，类型是StringBuilder。
        StringBuilder res = new StringBuilder();
		// 从row开始，遍历rows每一个元素
        for(StringBuilder row : rows) 
			// 在res不断添加row，构成最后的字符串
			res.append(row);
			// toString()：将任何对象转换成字符串表达形式 ，
			//  说直白点就是用文字描述这个对象里各来个变量是什么值，这个变量是什么类型的变量。
			// 转换成字符串表达形式。
        return res.toString();
    }
}
