  • 9_4 就餐人数：
         在为完成练习1胜9负而编写的程序中,添加一个名为服务数量的属性,并将其默认值设置为0,根据这个类创建一个名为 餐厅的实例;打印有多少人在这家餐馆就餐过,然后修改这个值并再次打印它。
         添加一个名为设置数量为()的方法,它让你能够设置就餐人数。调用这个方法并向它传递一个值,然后再次打印这个值。
         添加一个名为增加数量为()的方法,它让你能够将就餐人数递增。调用这个方法并向它传递一个这样的值:你认为这家餐馆每天可能接待的就餐人数。
class Restaurant():          #定义一个类
	def __init__(self,restaurant_name,cuisine_type):   #记住这里__init__的下划线是左右各两条__，而不是一条_
		self.restaurant_name = restaurant_name
		self.cuisine_type = cuisine_type
		self.number_served = 0                      #特殊方法下面的选项开头之前，都要添加“self”，代表默认
		
	def describe_restaurant(self):
		print(self.restaurant_name.title() + " is a very good restaurant.")
		print(self.cuisine_type.title() + " is very tasty food for most people.")

	def open_restaurant(self):      #不知道为什么，复制粘贴过来就出错 ==>复制到之前的地方，对比下，就很明显，发现： “语法错误”，第9行复制过来后，没发现，print结尾少了括号。
		print("Our restaurant is opening")

	def read_number_served(self):	  #打印一条支出多少人用餐的消息
		print("This restaurant has " + str(self.number_served) + " person will eat food.")
	
	def set_number_served(self,renshu) :        #通过方法设置就餐人数
		self.number_served = renshu
		
	def increment_number_served(self,renshu_2):           #增加就餐人数,人数不会减少回调
		if renshu_2 >= self.number_served:
			self.number_served = renshu_2
		else:
			print("You can't roll back number_served.")
		
		
		
restaurant = Restaurant('Good cooking','faguocai,guangdongcai')
restaurant.describe_restaurant()
restaurant.open_restaurant()

restaurant.read_number_served()

restaurant.number_served = 444      #直接修改属性值
restaurant.read_number_served()

restaurant.set_number_served(666)   #通过方法设置就餐人数
restaurant.read_number_served()

restaurant.increment_number_served(9)     #调用防止回调方法:人数只能递增。
restaurant.read_number_served()
