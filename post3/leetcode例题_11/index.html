盛最多水的容器

给你 n 个非负整数 a1，a2，…，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线， 垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线， 使得它们与 x 轴共同构成的容器可以容纳最多的水。

说明：你不能倾斜容器，且 n 的值至少为 2。

图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。

 

示例：

输入：[1,8,6,2,5,4,8,3,7] 输出：49

  • 分析 ： 关键在于找到挡板移动对面积影响的特点，没找到时很迷糊，找到后一个一个情况分析面积变化分类，就好理解了。

    即：向内移动短板才有可能导致面积增加。

    向内移动长板，只可能导致面积不变或减小。

    ===》 类似博弈论思维，只要一直移动短挡板就可以了，这样才可能发现最大面积是多少。

public  class Solution{
	public static int maxArea(int[] str) {
		int i = 0 ;
		int length_1 = str.length;
		int max_area = 0 ;   //   初始化最大面积
		int current_area = 0 ; //  初始化当前面积
		int j = length_1 - 1 ; // 初始化数组末尾的序数
		// 关键在于找到了两个挡板之间的特点，分析移动不同挡板时，对面积造成的的影响，有什么根本的原理。
		while (i <= length_1 - 1 && i < j && j >=1  ) {
			// 判断哪个挡板短
			if (str[i] <= str[j])  {
				// j - i : 底的长度 ；  h : 最矮的那个挡板
				current_area = (j - i ) * str[i] ;
				i++;
			} else { 
				current_area = (j - i ) * str[j];
				j-- ;
			}
			// 替换最大的面积 
			if (current_area > max_area) {
				max_area = current_area ;
			}
		}
		return max_area;
	}
	public static void main(String[] args)   {    
		int[] str_1 = new int[] {1,8,6,2,5,4,8,3,7} ;
		int result = 0;
		result = maxArea(str_1) ;
		System.out.println("result = " + result);
	}
}
