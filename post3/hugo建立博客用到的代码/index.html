hugo建立博客用到的代码

前几天看了b站的程序羊的教学视频，学会了搭建个人博客。配置完git和hugo环境后，用到了很多git命令，现在在这里把后几步的git命令记录在这篇博客当中

建立好hugo和git环境后的前提下,在cmd中

1:输入代码

hugo new site myblog3 (myblog3是我自己的命名的一个例子)

2:输入代码

git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c (这里的theme主题可以自己在网站上找，我这里是程序羊推荐的)

再输入代码

hugo server -t m10c --buildDrafts（在本地host先建立博客）

3:输入代码

hugo new post3/blog3.md (创建 文件夹post（自己命名），同时在建立博客.md文件（blog3也是自己命名的）)

4:输入代码

hugo --theme=m10c --baseUrl="https://taliantiger.github.io/" --buildDrafts(用于建立public文件，其中"https://taliantiger.github.io/“填写你是自己之前建立的github仓库的地址)

5:输入代码

cd public (转到public这个仓库)

6:输入代码

git init (git基本操作）

7:输入代码

git add . (git基本操作）

8:输入代码

git commit -m "taliantigertest" （把public文件提交到github上,其中"taliantigertest"可以自己填写，用作本次上传的标注文字）

9:输入代码

git remote add origin https://github.com/taliantiger/taliantiger.github.io.git （把本地文件挂在远端）

10:如果出现错误（如下图，则输入）

git remote rm origin

11: 输入代码

git push -u origin master （开始启动push往github上推）
