回文数

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

示例 1:

输入: 121

输出: true

示例 2:

输入: -121

输出: false

解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。

示例 3:

输入: 10

输出: false

解释: 从右向左读, 为 01 。因此它不是一个回文数。

进阶:

你能不将整数转为字符串来解决这个问题吗？

  • 普通解法思路：简单分析回文数，了解它的特点。把整数转为数组， 比较首末是否相同。
// 自己想的简单解法。
public class Solution   {
	public static boolean isPalindrome(int x)  {
		boolean flag = true ;    // 初始化标识符
		int i = 0  ;          // 初始化整型变量 i 
		int j = 0;            // 初始化变量 j 
		int length_1 = 0;       // 初始化变量length_1

		// String.valueOf(x) : 把整型x转化为字符串型 。
		 String str = String.valueOf(x);        // 把整型数字转为字符串型
		//  String str = Integer.toString(x);        // 把整型数字转为字符串型

		char[] chars = str.toCharArray();    // 把字符串型数字转为char型数组数组
		length_1 = chars.length;            // 定义数组的大小
		 j = length_1 - 1;
		 if (chars[0] == '-')  {           // 如果整数是负数
			flag = false;                    // 并非回文数
		 }
       // 注意这里条件限制要为i <= length_1 / 2 - 1 ;而不是< ,只有小于号时，会出现数字位数只有2时，直接跳过for循环的错误。
		// for (i = 0; i < length_1 / 2 - 1 ; i++ )   {   
		for (i = 0; i <= length_1 / 2 - 1 ; i++ )   {    
			if (chars[i] == chars[j] )  {     // 如果首位相等
				j--;                                 
				continue;                       // 则不断分析下一个元素
			}
			else  {
				flag = false;
				break;
			}
		}
		return flag;
	}

	public static void main(String[] args)   {
        int num_1 = 10;
		boolean flag = true;
		flag = isPalindrome(num_1) ; 
		System.out.println(flag);
	}  
}
