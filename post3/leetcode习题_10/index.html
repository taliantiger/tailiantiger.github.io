正则表达式匹配

给你一个字符串 s 和一个字符规律 p，请你来实现一个支持 ‘.’ 和 ‘*’ 的正则表达式匹配。

‘.’ 匹配任意单个字符

‘*’ 匹配零个或多个前面的那一个元素

所谓匹配，是要涵盖 整个 字符串 s的，而不是部分字符串。

说明:

s 可能为空，且只包含从 a-z 的小写字母。

p 可能为空，且只包含从 a-z 的小写字母，以及字符 . 和 *。

示例 1:

输入:

s = “aa”

p = “a”

输出: false

解释: “a” 无法匹配 “aa” 整个字符串。

示例 2:

输入:

s = “aa”

p = “a*”

输出: true

解释: 因为 ‘*’ 代表可以匹配零个或多个前面的那一个元素, 在这里前面的元素就是 ‘a’。因此，字符串 “aa” 可被视为 ‘a’ 重复了一次。

示例 3:

输入:

s = “ab”

p = “.*”

输出: true

解释: “.” 表示可匹配零个或多个（''）任意字符（'.'）。

示例 4:

输入:

s = “aab”

p = “cab”

输出: true

解释: 因为 ‘*’ 表示零个或多个，这里 ‘c’ 为 0 个, ‘a’ 被重复一次。因此可以匹配字符串 “aab”。

示例 5:

输入:

s = “mississippi”

p = “misisp*.”

输出: false

// 代码虽然短，但是十分精炼，十分抽象，提到了正则表达式，还有递归调用自己本身的方法。
// 注意是要完全匹配，而不是部分匹配
// 例如 s：aab；  p ： cba*b  ==》 结果：不匹配，palse。
// 因为：不是p中包含了s就行，而是p和s要对应。c*a*b可以看做是c0a1b=>aab，楼主的cba*b，不管怎样看，前面的cb都是不匹配的，cb都不会被消掉。
// 强调p:“不多也不少” 
public  class Solution {
	// text 对应题目s，pattern 对应题目p
    // 自己构建一个方法，名为isMatch,之后的代码，会不断递归调用自己这个isMatch方法。
	public static  boolean isMatch(String text, String pattern)  {
		// 判断pattern是否是空字符串
		if (pattern.isEmpty() ) { 
			// 如果text也是空字符串，则返回true；否则返回false;
			 return text.isEmpty();
		}

		// charAt() 方法用于返回指定索引处的字符。索引范围为从 0 到 length() - 1。At()代表"在",即强调索引所在的位置
		// first_match：确保两组字符串，头个元素都相等且为非空；
		// 分析第一个匹配的布尔值：首先text不能是空字符串 && (pattern 的第0个元素 和 text的第0个元素要相等，或patter 的第0个元素是万能变换符——'.'）。
		boolean first_match = (!text.isEmpty() && (pattern.charAt(0) == text.charAt(0) || pattern.charAt(0) == '.') );
		
		// 当pattern的长度大于等于2，同时“pattern[1]”为 '*' 时,则：
		if (pattern.length() >= 2 && pattern.charAt(1) == '*' )	{   
			// substring()： 截取字符串
			// substring(2)： 截取字符串序数为2（包括序数2） 之后的所有字符
			// isMatch (text, pattern.substring(2) ）：整体分析test; 判断字符串pattern.substring(2) 是否整体包含了子字符串 text ，同时pattern.substring(2)“不多也不少"
			// 当调用isMatch方法时，自动会启用Java中正则表达式的特性，对字符串进行分析
			return ( isMatch (text, pattern.substring(2) )  || 
						// isMatch(text.substring(1), pattern)  : 单独分析text.substring(1)；判断字符串pattern 是否整体包含了子字符串text.substring(1) ，同时pattern“不多也不少”
				         (first_match && isMatch(text.substring(1), pattern)  )  ) ;
		}  else { 
			// isMatch( text.substring(1), pattern.substring(1) ) ： 判断字符串pattern.substring(1)是否包整体含子字符串text.substring(1) ，同时pattern.substring(1) 不多也不少” 
			return first_match && isMatch( text.substring(1), pattern.substring(1)  ) ;
		}
	}
	public static void main(String[] args) {
	  boolean flag = false ;
	  String s = "aab" ;
	  String p = "c*a*b"  ;
	      flag = isMatch(s , p );
	  System.out.println("flag = " + flag);	
	}
}
