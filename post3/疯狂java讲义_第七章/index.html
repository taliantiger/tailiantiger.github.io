7.1.2 使用Scanner获取键盘输入

  • 正则表达式：实质就是一种字符串匹配的模式(pattern)，可以用来检查一个串是否含有某种子串， 将匹配的子串替换或从某个串中取出符合某个条件的子串等。

    正则表达式实例: 一个字符串其实就是一个简单的正则表达式，例如 Hello World 正则表达式匹配 “Hello World” 字符串。

    .（点号）也是一个正则表达式，它匹配任何一个字符如：“a” 或 “1”。

  • 声明一个对象，同时使对象实例化

    var sc = new Scanner(System.in); // System.in 代表用户手动输入

  • Scanner类提供了两个方法来扫描输入

    hasNextXxx() :是否还有下一个输入项。(has表示疑问的语气，即“是否”。)

    nextXxx() : 获取下一输入项。
