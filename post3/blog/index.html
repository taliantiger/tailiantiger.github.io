论述如何在电脑本地git里，给github上传文件

1.创建好github文件库，打开文件库，出现文件库地址备用

2.接下来进行本地操作，在本地创建一个文件夹，右击，选择Git Bash Here,进入git界面

3.再然后，在界面输入如下代码，把github上的仓库克隆到本地

git clone https://github.com/Talian000001/photo.io （“https://github.com/Talian000001/photo.io"替换成你之前复制的地址）

4.执行完第3步后，你的本地项目文件夹下面会多出个文件夹，我们把本地项目下所有文件（新增加的文件夹不用）都移动到新增的文件夹下

5.接着输入命令 cd photo.io ，进入 photo.io 文件夹

6.再然后，一次输入如下代码

git add . (用于把photo文件夹下的文件都添加进来)

git commit -m “abc” (相当于一次提交备注，所有这次提交地文件，都会有这个备注)

git push -u origin master (把本地仓库push到github上，这个步骤要输入你的账号和密码)
